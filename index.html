¿Cómo puedo registrarme para el congreso?

Puede registrarse en línea a través de nuestra página web llenando sus datos en el Formulario para contactarno con usted, o en persona en el lugar del evento.

¿Cuál es la tarifa de registro para el congreso?

La tarifa de registro varía dependiendo del tipo de participante y la fecha de registro. Por favor consulte nuestra página web para obtener más detalles.

¿Hay alguna tarifa especial para estudiantes?

Sí, ofrecemos una tarifa especial para estudiantes. Por favor consulte nuestra página web para obtener más detalles.
